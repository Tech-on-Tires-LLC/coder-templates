# Coder Templates

This repository contains Terraform templates for provisioning [Coder](https://coder.com/) workspaces on various providers.

## Available Templates

| Template | Provider | Description | Path |
|----------|----------|-------------|------|
| **Vultr** | Vultr | Provisions a Coder workspace on a Vultr VPS instance. | [`vultr/`](./vultr) |

## Usage

1.  Navigate to the directory of the template you wish to use.
2.  Follow the `README.md` instructions within that directory to configure and upload the template to your Coder deployment.
